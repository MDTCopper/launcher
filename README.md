# Copper Launcher

基于 **Flutter** 跨平台开发的 [Mindustry](https://github.com/Anuken/Mindustry) 游戏启动器，目前处于开发阶段

## 可运行平台

- Windows
- Linux
- Android

## 核心功能

- 多版本管理：一键切换、管理 Mindustry 游戏版本和存档
- Mindustry下载：可下载最新版本和过往游戏版本
- 社区资源下载：支持模组、地图、蓝图等资源浏览与下载
- 外部资源导入：可手动导入本地游戏本体、模组、蓝图等外部资源
- 一键更新游戏和模组版本
- 内置实用小工具集合

## 提醒

当前游戏资源和社区资源均来自github，国内访问受限
